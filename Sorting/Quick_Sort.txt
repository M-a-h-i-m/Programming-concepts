Quick Sort 
------------------

⚙️ What is Quick Sort?
Quick Sort is a divide-and-conquer sorting algorithm that works by selecting a pivot element and partitioning the array so that:

01. Elements less than the pivot go to the left
02. Elements greater than the pivot go to the right

After partitioning, Quick Sort is recursively applied to the left and right subarrays until the entire array is sorted.

🧩Quick Sort is in-place, but not stable.
Example:

arr = [8, 3, 1, 7, 0, 10, 2] 

Pivot chosen: last element 2

Partition:

♦️ Left: [1, 0]
♦️ Pivot: 2
♦️ Right: [8, 3, 7, 10]

Recursively sort left → [0, 1]
Recursively sort right → [3, 7, 8, 10]

Combine → [0, 1, 2, 3, 7, 8, 10]

🧠 Algorithm Intuition:
01. Pick a pivot (last element or random).
02. Partition array into two parts: ≤ pivot, > pivot.
03. Recursively sort left and right partitions.
04. Combine the results.


💻 Example Step-by-Step
Start: [8, 3, 1, 7, 0, 10, 2]

♦️ i = pivot = 2

Left: [1,0], Right: [8,3,7,10]

♦️ Sort left [1,0] → [0,1]

♦️ Sort right [8,3,7,10] → [3,7,8,10]

♦️ Combine → [0,1,2,3,7,8,10]



🚀 Time and Space Complexity

Time:
01. Best/Average: O(n log n)
02. Worst: O(n²) (bad pivot selection)

Space:
01. O(log n) due to recursion stack
