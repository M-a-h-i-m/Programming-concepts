
Merge Sort 
------------------
⚙️ What is Merge Sort?
Merge Sort is a divide-and-conquer sorting algorithm that divides the array into two halves, recursively sorts each half, and then merges the two sorted halves into one sorted array.

It is stable, but requires extra space for merging.


🧩 Example:
arr = [8, 3, 1, 7, 0, 10, 2]

01. Divide: [8,3,1,7] & [0,10,2]
02. Divide again: [8,3] [1,7] & [0,10] [2]
03. Sort smallest units: [8,3] → [3,8], [1,7] → [1,7], [0,10] → [0,10], [2]
04. Merge: [3,8] + [1,7] → [1,3,7,8], [0,10] + [2] → [0,2,10]
05. Merge final: [1,3,7,8] + [0,2,10] → [0,1,2,3,7,8,10]


🧠 Algorithm Intuition:
01. Split array into two halves.
02. Recursively sort each half.
03. Merge the two sorted halves together.
04. Repeat until fully sorted.



💻 Example Step-by-Step
Start: [8, 3, 1, 7, 0, 10, 2]

♦️ Divide → [8,3,1,7] & [0,10,2]
♦️ Divide → [8,3] [1,7] & [0,10] [2]
♦️ Sort smallest units → [3,8], [1,7], [0,10], [2]
♦️ Merge → [1,3,7,8], [0,2,10]
♦️ Merge final → [0,1,2,3,7,8,10]


🚀 Time and Space Complexity
01. Time: O(n log n) (always)
02. Space: O(n) (extra array needed for merging)
